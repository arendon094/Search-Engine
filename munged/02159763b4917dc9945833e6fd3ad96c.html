Yass (software)

From Wikipedia, the free encyclopedia

For other uses, see Yass.

YASS[1][2] is a public domain, pairwise sequence alignment software for nucleotide sequences. This program accepts sequences in FASTA format and the output format includes the BLAST tabular output. YASS uses several transition-constrained seeds that allow to considerably improve the sensitivity. The program can be used through the YASS web server or as a standalone program which can be installed with the source code.
