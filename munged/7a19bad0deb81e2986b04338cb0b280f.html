.ics

From Wikipedia, the free encyclopedia

.ics is a computer filename extension used to designate a file containing (an arbitrary set of) calendaring and scheduling information consistent with iCalendar. Some of the programs that run with the .ics format include Apple Inc.'s iCal and Microsoft's iCalendar.
