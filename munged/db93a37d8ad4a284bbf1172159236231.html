Criminal Code of Belarus

From Wikipedia, the free encyclopedia

The Criminal Code of the Republic of Belarus contains the fundamental laws to announce what is considered illegal to perform inside Belarus. Passed in 1999, several of these laws were carried over from laws passed in 1960 as the Byelorussian SSR. In the 1999 edition, the code contains passages of crimes that can be issued capital punishment upon sentencing.
